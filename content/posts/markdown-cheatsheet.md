写博客不需要背完整语法，掌握下面这些就够用。

## 标题

```md
# 一级标题
## 二级标题
### 三级标题
```

## 强调

- *斜体*：`*斜体*` 或 `_斜体_`
- **粗体**：`**粗体**`
- `行内代码`：用反引号包裹

## 列表

无序列表：

- 苹果
- 香蕉
- 橙子

有序列表：

1. 打开编辑器
2. 新建 `.md` 文件
3. 写下 frontmatter 与正文

## 代码块

指定语言可获得高亮（取决于渲染器）：

```ts
export function greet(name: string) {
  return `Hello, ${name}!`;
}
```

## 引用

> 好的文档像好的代码：清晰、克制、可维护。

## 链接与图片

```md
[Next.js 文档](https://nextjs.org/docs)
![替代文本](/path/to/image.png)
```

## 表格

| 语法 | 效果 |
|------|------|
| `**粗体**` | **粗体** |
| `` `code` `` | `code` |

把这些组合起来，就能写出结构清晰的技术文章。
